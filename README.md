# HR Analytics Tableau Dashboard

## Project Overview
This project analyzes employee attrition and workforce trends using Tableau. The dashboard helps identify key factors affecting employee turnover such as department, overtime, gender, and job satisfaction.
## Tools Used
- Tableau Public
- Excel / CSV
- HR Analytics Dataset
## Dashboard Features
- Employees by Department
- Gender Distribution
- Attrition by Department
- Overtime Impact on Attrition
- Job Satisfaction Analysis
## Key Insights
- Employees working overtime show higher attrition risk.
- Research & Development has the highest employee count.
- Job satisfaction has a clear impact on employee retention.
- Tableau filters allow interactive workforce analysis.
